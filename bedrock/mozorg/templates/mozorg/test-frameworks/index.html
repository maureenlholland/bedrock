What are we talking about?

Web Components use three main technologies to create custom, reusable, framework-agnostic HTML elements: custom elements API, shadow DOM, and HTML templates / slots.

Web Component frameworks abstract away the boilerplate of Web Component development and include helpers for things like reactivity. Ex: Lit

Javascript frameworks encourage app-based web development with dynamic, composable, interactive components. They can be used with or without Web Components. Ex: Svelte
