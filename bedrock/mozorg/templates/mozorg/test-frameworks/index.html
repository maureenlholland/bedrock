What is a front-end framework?

Web Components use three main technologies to create custom, reusable, framework-agnostic HTML elements: custom elements API, shadow DOM, and HTML templates or slots.

Web Components are still pretty low level and it would be a steep learning curve to use them without a framework at this time.

Web Component frameworks abstract away the boilerplate of Web Component development and include reactivity but fall short on data-binding and state management when compared to more established Javascript frameworks.

The increased power of Javascript frameworks comes with a heavier bundle size cost.
